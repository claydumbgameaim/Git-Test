How to Run the Simulation:

1) Open up the command prompt.

2) Go to the directory in which all the source code is. There should be a 'wifi.jar'.

3) Make sure you have java. Type in 'java -jar wifi.jar'

4) You should be prompt with a question asking for asynchronous or synchronous. Pick a choice.

5) Type in the number of nodes you want in the network.

6) After that simulation ends, It will ask if you want to exit.
